## How to Push Changes:

### First Time Push:
1. **Stage your changes:**
   ```bash
   git add .
   ```

2. **Commit your changes:**
   ```bash
   git commit -m "Your commit message"
   ```

3. **Push to GitHub:**
   ```bash
   git push
   ```
   
4. **Windows will show a popup** asking for your GitHub credentials
   - Enter your GitHub username
   - Enter your GitHub password or Personal Access Token
   - Credentials will be saved automatically

### Future Pushes:
After the first authentication, you can simply:
```bash
git add .
git commit -m "Your commit message"
git push
```
